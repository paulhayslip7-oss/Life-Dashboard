How to Connect APIs

Each integration uses its official API. You'll need to:

  1. Spotify: Create an app at developer.spotify.com → get Client ID + Secret → paste below.
  2. Gmail: Enable Gmail API in Google Cloud Console → create OAuth credentials → paste Client ID below.
  3. Calendar: Same Google Cloud project as Gmail → enable Calendar API → reuse credentials.

All keys are stored in your browser only (localStorage). Nothing is sent to any server.

🎵 Connect Spotify
1. Go to developer.spotify.com/dashboard
2. Create an app → copy your Client ID
3. Add http://localhost as a Redirect URI
4. Paste the Client ID below
📧 Connect Gmail
1. Go to console.cloud.google.com
2. Create project → Enable Gmail API
3. Create OAuth 2.0 credentials → copy Client ID
4. Add this file's path as authorized origin
📅 Connect Google Calendar
Uses the same Google OAuth Client ID as Gmail.
If you've already set up Gmail, just paste the same Client ID here.

Enable the Google Calendar API in your Cloud project.
